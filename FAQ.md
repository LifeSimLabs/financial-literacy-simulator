# Frequently Asked Questions

**Q: What technology stack will be used?**
A: We are currently in the Research and Design phase. The technology stack will be decided collaboratively through architecture discussions. See our Discussions tab to participate.

**Q: I don't know how to code. Can I still contribute?**
A: Absolutely! We need financial experts to help define formulas, designers for the UI/UX, educators to help with scenarios, and writers for documentation.

**Q: How realistic will the simulation be?**
A: We aim for a high degree of realism regarding financial mechanics (compound interest, taxes, inflation), but we will balance this with gameplay and accessibility so it doesn't feel like a spreadsheet.

**Q: Will it support different countries' tax and financial systems?**
A: Initially, we will focus on a generalized or US-centric model for the MVP, but the architecture will be designed to be modular so that localized systems can be added later.\n
